Installation:

Begin by running the following command in the python terminal:
pip install pytest


Usage:

Consider the following function definition.
def capital_case(x):
return x.capitalize()

The core of writing a tests is essentially checking if the output matches what is expected. All tests must begin with test_ or end in _test in their file names.

An appropriate test is as follows.
def test_capital_case():
assert capital_case('semaphore') == 'Semaphore'

Run pytest by writing pytest into the python terminal


Testing Importance:

Testing is necessary for any robust application, and unit testing helps in multiple ways. One, unit testing targets specific parts of your application, allowing for more specific tests. Two, unit testing is far faster than manual evaluation, allowing for a higher quantity of tests. Three, unit testing is consistent. After writing the tests, there is no chance for human error evaluating incorrectly.
